Regex:

\	-	escape character
.	-	any character except new line
\d	-	any digit (0-9)
\D	-	any non digit
\w	-	word character (a-z, A-Z, 0-9, _)
\W	-	non word character
\s	-	any white space( space, tab, newLine)
\S	-	non white space character

\b	-	word Boundary
^	-	Beginning of a string
$	-	End of the String
[]	-	character set(dont have to escape inside character set) 	[12]-	1 or 2 		[1-9]-	1 to 9
[^]	-	Negate the set		[^a-zA-Z]	-	not in a-z or A-Z

* and + are greedy operators, add ? to make it non greedy

"\\bWHERE\\b.+(GROUP BY|HAVING|JOIN|ORDER BY|LEFT JOIN|RIGHT JOIN|FULL JOIN|SELF JOIN|INNER JOIN|WITH|UNION)?",

Quantifiers:

*	-	0 or more
+	-	1 or more
?	-	0 or 1

{3}		exact Number count
{3,9}	range of Numbers 3 count to 9 count		\d{3}	=	\d\d\d

()	-	group
(r|s|rs)	-	r or s or rs
(1)(2)(3) - first group, second group, thrid group
Flags : 
/ - Regex starts and ends with /
/pattern/g - global match
/pattern/i - case insensitive
/pattern/gi - global case insensitive match.



.       - Any Character Except New Line
\d      - Digit (0-9)
\D      - Not a Digit (0-9)
\w      - Word Character (a-z, A-Z, 0-9, _)
\W      - Not a Word Character
\s      - Whitespace (space, tab, newline)
\S      - Not Whitespace (space, tab, newline)

\b      - Word Boundary
\B      - Not a Word Boundary
^       - Beginning of a String
$       - End of a String

[]      - Matches Characters in brackets
[^ ]    - Matches Characters NOT in brackets
|       - Either Or
( )     - Group

Quantifiers:
*       - 0 or More
+       - 1 or More
?       - 0 or One
{3}     - Exact Number
{3,4}   - Range of Numbers (Minimum, Maximum)
